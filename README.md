# cpp-transport-catalogue
Транспортный справочник

Система умеет хранить автобусные маршруты и остановки, находить кратчайшие пути,рассчитывать время поездки 
и визуализировать карту маршрутов в формате SVG.

Ключевые возможности
 * Поиск маршрутов: Нахождение кратчайшего пути между остановками с учетом времени ожидания и движения.
 * Визуализация: Генерация карты маршрутов в формате SVG с гибкой настройкой цветов и размеров.
 * Гео-расчеты: Вычисление реальной дистанции пути по координатам и сравнение с дорожной дистанцией.
 * JSON-интерфейс: Ввод данных и получение статистики происходит через стандартные потоки ввода/вывода (stdin/stdout).

Архитектура
Проект разделен на логические модули:
 * TransportCatalogue — основная база данных (остановки и маршруты).
 * JsonReader — парсинг входных данных и формирование ответов.
 * MapRenderer — подсистема отрисовки карты в SVG.
 * TransportRouter — построение графа и поиск кратчайших путей с помощью алгоритма Дейкстры.
 * RequestHandler — фасад, связывающий базу данных и визуализатор.

Сборка
Проект собирается с помощью CMake. Для работы требуется компилятор с поддержкой стандарта C++17 и выше.
mkdir build && cd build
cmake ..
cmake --build 

Использование
Программа работает в режиме потоковой обработки. Она ожидает на вход JSON-объект, содержащий настройки базы данных 
(base_requests), настройки отрисовки (render_settings) и запросы к справочнику (stat_requests).
Запуск:
./transport_catalogue < input.json > output.json

Пример входных данных:
{
  "base_requests": [...],
  "render_settings": {...},
  "routing_settings": {"bus_wait_time": 6, "bus_velocity": 40},
  "stat_requests": [...]
}

Структура проекта
 * transport-catalogue/ — исходный код всех модулей (.h и .cpp файлы).
 * CMakeLists.txt — конфигурация сборки
